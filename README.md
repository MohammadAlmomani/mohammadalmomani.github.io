# Mohammad M. Al-Momani

Power systems researcher working on optimal PMU placement, power system oscillation detection, PV modeling, transient stability prediction, and AI-based monitoring of modern power systems.

## Research Areas

- Optimal PMU placement and power system observability
- Low-frequency oscillation detection and ringdown analysis
- PV array modeling and optimization
- Transient stability prediction using artificial neural networks
- DERs, renewable energy, and inverter-dominated power systems
